# Bloomberg Terminal (bloomberg-terminal)
Bloomberg Terminal is a computer software system provided by Bloomberg L.P. that enables professionals in the financial service sector and other industries to access the Bloomberg Professional service through which users can monitor and analyze real-time financial market data and place trades.

**URL:** [Visit APIs.json URL](https://www.bloomberg.com/professional/solution/bloomberg-terminal/)
